# Data acquisition

The public reproducibility repository intentionally contains no third-party
empirical data. Koren (2018) and Lelkes, Sood, and Iyengar (2017) data remain
subject to original-author/provider redistribution terms. See each replication
directory's `DATA.md` for the required filenames, checksums, sample-size
expectations, and validation command.

Do not place raw data under version control. Configure `SPLIV_KOREN_DATA` or
`SPLIV_LELKES_DATA` locally and run the corresponding `check_data.R` script.
